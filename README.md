# 自制线程池

## 实现功能
+ 开启线程时，能指定最小线程数、最大线程数、最大任务数、管理者检查间隔
+ 添加任务时、可以传入参数（目前不支持获取返回值，请使用引用传参）
+ 设置管理者线程，可以自动增加和减少线程
+ 可以手动释放线程，安全回收资源、并且不会丢失任务
