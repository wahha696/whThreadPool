# 自制线程池

## 线程池支持功能：

+ 启动线程池，自定义线程池的最小和最大线程数量
+ 添加任务，可以添加任务参数（目前不支持获得返回值，可以使用引用传参）
+ 具有管理者线程，自动控制线程数量
+ 提供手动释放线程池接口，安全释放
